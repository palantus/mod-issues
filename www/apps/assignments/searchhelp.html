General: The search will take place while typing. If you separate keywords with space, it means that the results needs to contain both conditions (i.e. AND). Pipe (|) is OR, thus the results needs to contain one of the keywords. A keyword can be negated by using exclamation mark (!) before a keyword.

Advanced search: Typing colon before a term signals that it is a special term. Currently the following are supported:

  • :all - all issues (including closed)
  • :closed - issue is closed
  • :open - issue is not closed
  • :dev - issue has status created or in development
  • :def - your default assignment
